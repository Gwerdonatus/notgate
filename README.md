# NOTGATE

Pixel-accurate reconstruction of a luxury real estate development website.

## Tech Stack

- Next.js 15
- TypeScript
- Tailwind CSS
- Framer Motion
- Lucide Icons

## Getting Started

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000)

## Pages

- `/` - Homepage (Hero, About, Partners, Team, Projects, Clients, Stats, CTA)
- `/company` - About Company (Values, Stats, Services, Partners, Milestones)
- `/projects` - Featured Projects
- `/contact` - Contact (Form, Presence, Map)
